# NET2201 Computer Networks Group Assignment Pack

This workspace contains a ready-to-use planning and implementation pack for the NET2201 group assignment.

## Files

- `docs/workplan-checklist.md` - week-by-week checklist and evidence tracker.
- `docs/network-design.md` - proposed topology, VLAN/IP plan, port map, routing plan, security plan, and test plan.
- `docs/packet-tracer-and-lab-build-guide.md` - step-by-step Packet Tracer and lab build instructions.
- `docs/report-template.md` - report scaffold matching the assignment sections.
- `docs/video-and-presentation-guide.md` - 3-4 minute demo script and 15-minute presentation structure.
- `diagrams/topology.mmd` - Mermaid topology diagram source.
- `configs/R1-INTERNAL.ios` - internal router configuration.
- `configs/R2-ISP.ios` - ISP/external router configuration.
- `configs/SW1-DIST.ios` - first switch configuration.
- `configs/SW2-ACCESS.ios` - second switch configuration.

## Recommended Workflow

1. Review `docs/network-design.md` with your group.
2. Build the topology in Cisco Packet Tracer using the port map.
3. Paste the matching configs into each device.
4. Run every command listed in `docs/workplan-checklist.md`.
5. Save screenshots and outputs into your report.
6. Repeat the same design in the physical lab and record evidence.

Adjust interface names if your Packet Tracer or lab equipment uses different ports.
